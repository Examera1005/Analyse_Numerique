# Analyse Numérique

Ce dépôt contient une retranscription en **LaTeX** de notes de cours d'Analyse Numérique de printemps 2026.

## Contenu

- `main.tex` : point d'entrée du document
- `prembule.tex` : préambule, packages et styles
- `warning.tex` : avertissement et informations de contact
- `chapters/week1.tex` à `chapters/week14.tex` : chapitres du cours
- `Cover_Analyse_Numérique.png` : image de couverture

## Compilation

Le document est conçu pour une compilation LaTeX moderne (notamment avec `fontspec`).
Exemple de commande locale :

```bash
lualatex main.tex
```

Selon votre installation, plusieurs passages peuvent être nécessaires pour stabiliser la table des matières.

## Avertissement

Le contenu provient de notes de cours retranscrites, avec une mise en forme assistée par IA.
Des erreurs peuvent subsister. En cas de problème, ouvrez une issue sur ce dépôt.
